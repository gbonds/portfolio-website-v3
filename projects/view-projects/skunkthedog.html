Skunk the Dog

Lifestyle

Website Design & Development
HTML | CSS | Git & Github
View Github Repo

I designed and developed this concept website as a way to explore playful, interactive web design for a "pet influencer." The goal was to engage site visitors while also providing a basic level of information and establishing a brand identity.

I made use of very basic colors and a tongue-in-cheek decorative font. To drive visitor engagement, each page has some sort of call-to-action. I made use of slight animations to encourage a delightful experience on the site.
